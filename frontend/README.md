# OMA Frontend (React 17, Node 18.16.1)

## Run locally

1. Install:
   ```bash
   npm install
   ```
2. Configure API endpoint:
   ```bash
   cp .env.example .env
   ```
3. Start:
   ```bash
   npm run dev
   ```

UI coverage implemented:
- Main tab: **Order Management Automation**
- Subtabs: **Action Required** and **All Orders**
- Department status + per-department download actions
- Timeline view
- Ignore action available only in Action Required
- Settings → Configuration: printer management and department assignment
